# pxeger's notes system

Web-based notes application which saves to a plain text file of bullet points. The idea is that you can edit the text
file directly in your text editor of choice, rather than using the web interface, if you prefer.

A live demo is available at [pn-demo.pxeger.com](https://pn-demo.pxeger.com).

## Features
- Live editing over WebSockets
- Light and dark themes
- Core functionality works without JavaScript
- Drag and drop node reorganisation
